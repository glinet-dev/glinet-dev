## About Me

I am a Senior React / Frontend Engineer with professional experience building healthcare SaaS interfaces, dashboards, forms, role-based workflows, and API-integrated frontend systems.

My strongest area is frontend product engineering: turning requirements into clean UI flows, reusable components, reliable state management, dashboard experiences, loading/error/empty states, testable code, and sprint-ready delivery.

I also bring Scrum and product ownership experience through backlog refinement, user stories, acceptance criteria, sprint planning, UAT support, release readiness, stakeholder communication, and mentoring junior developers.

Current direction:

- Strong professional base: React, TypeScript, JavaScript, Redux Toolkit, REST APIs, dashboards, SaaS UI, Agile/Scrum.
- Product strength: healthcare SaaS workflows, acceptance criteria, UAT, stakeholder coordination, release readiness.
- Building toward: Django REST Framework, PostgreSQL/SQL, BI dashboards, cloud deployment, and practical AI-assisted product features.

Target roles:

`Senior React Developer` · `Senior Frontend Engineer` · `Frontend Technical Lead` · `Product Engineer` · `Technical Product Owner` · `Frontend-led Full-Stack Developer`
